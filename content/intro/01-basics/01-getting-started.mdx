### A Simple C++ Program

Here is the standard "Hello World" template in C++:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << "Hello, BdOI!" << endl;
    return 0;
}
```

<Warning>
Always make sure to return 0 from your `main()` function!
</Warning>
